# Swift-chain-analysis

## How to use


## Download

```shell
git clone -b swift-chain-analysis https://gitlab.com/storswiftlabs/swiftos-ui.git
```

## Install it and run:

```shell
cd swift-chain-analysis/ 
yarn install
yarn dev
```

### build

```shell
yarn build
```
